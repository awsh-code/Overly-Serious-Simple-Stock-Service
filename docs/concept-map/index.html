üîÑ Request Processing Flow

  1. Request Arrival: HTTP request hits the Gorilla Mux router
  2. Middleware Processing: Logging and metrics collection
  3. Route Handling: Appropriate handler based on URL pattern
  4. Cache Lookup: Check if data exists in memory cache
  5. Circuit Breaker Check: Verify external API availability
  6. External API Call: Fetch data from Alpha Vantage (if needed)
  7. Response Processing: Format data and calculate averages
  8. Cache Storage: Store results for future requests
  9. Metrics Update: Update Prometheus metrics
  10. Response Delivery: Send JSON response to client

üìà Traffic

How much demand is placed on your system, measured by requests per second.
